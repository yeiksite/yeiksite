McAfee VirusScan

McAfee VirusScan es uno de los antivirus más populares del mercado, creado y mantenido por la empresa McAfee, conocida anteriormente por Network Associates. El VirusScan Plus fue creado para uso doméstico; el VirusScan Enterprise para uso en pequenas y medias empresas. En complemento de estos, existen otros produtos McAfee como McAfee Internet Security, que oferece seguridad mayor para computadoras con acceso a Internet, y el McAfee Total Protection (All-in-one) que oferece una protección completa para el computador. Tanto el McAfee Internet Security como McAfee Total Protection poseen como componente el VirusScan, además, Internet Security posee McAfee SiteAdvisor.

Sistemas operativos admitidos

  • Windows XP SP3 o superior (32 bits)
  • Windows Vista SP1 (32 y 64 bits)
  • Windows 7 (32 y 64 bits)
  • Windows 8 (32 y 64 bits)
  • 512 MB de RAM para XP
  • 2 GB de RAM para Vista y Windows 7
  • 500 MB de espacio libre en disco
  • Procesador a 1 GHz

Es necesaria una conexión a Internet

  • Se recomienda una conexión de alta velocidad

Lista de navegadores compatibles con antiphishing

  • Internet Explorer 7.0 o posterior
  • Firefox 4.0 o posterior
  • Google Chrome

Diferentes programas de correo electrónico compatibles

  • POP3: Windows Mail, Outlook Express, Outlook, Netscape, IncrediMail, Thunderbird, Becky, Shuriken
  • MAPI: Outlook
  • Web: MSN/Hotmail o cuenta de correo electrónico con acceso POP3

Necesidades para la barra de herramientas Antispam

  • Windows Mail u Outlook Express 6.0+
  • Outlook XP+
  • Thunderbird 5.0+
  • McAfee Anti-Spam también es compatible con otros clientes de correo electrónico POP3 (sin SSL) y cuentas web

Lista de programas de mensajes instantáneos compatibles

  • AOL Instant Messenger 2.1+
  • Yahoo Messenger 4.1+
  • Microsoft Windows Messenger 3.6+
  • MSN Messenger 6.0+
